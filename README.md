# 東京スター銀行 FAQ（静的ページ）— Kiro Spec駆動ハンズオン用

静的なFAQページです。ここにKiroのSpec駆動開発でチャット機能を追加するハンズオンを行います。

## セットアップ

```bash
git clone https://github.com/TKssk01/kiro-handson-faq.git
cd kiro-handson-faq
npm install
npm run dev
```

ブラウザで http://localhost:5173 にアクセスすると、静的なFAQページが表示されます。

## ハンズオン

このプロジェクトをKiroで開いて、Spec Modeで以下のようなプロンプトを投げてみてください:

> 「この静的FAQページにAIチャット機能を追加したい。ユーザーが自然言語で質問すると、FAQデータから最も関連性の高い回答を返すチャットウィンドウを画面右下に追加してください。」
