# 要件定義書 — 東京スター銀行 FAQサイト

## 1. プロジェクト概要

東京スター銀行の公式Webサイト内に、お客さま向けFAQページを構築する。

## 2. 目的

- お客さまからの問い合わせ件数を削減する
- 24時間いつでもお客さまが自己解決できる環境を提供する
- コールセンターの負荷を軽減する

## 3. ターゲットユーザー

- 東京スター銀行の個人顧客（既存・新規）
- 年齢層: 20代〜70代
- デバイス: PC（60%）、スマートフォン（35%）、タブレット（5%）

## 4. 機能要件

### 4.1 FAQカテゴリ表示
- カテゴリ別にFAQを分類して表示する
- カテゴリ: 住宅ローン、カードローン、口座・預金、インターネットバンキング
- カテゴリタブで切り替え可能

### 4.2 FAQ検索
- キーワード検索でFAQを絞り込める
- 検索対象: 質問文、回答文

### 4.3 FAQアコーディオン
- 質問をクリックすると回答が展開される
- 再度クリックで閉じる

### 4.4 お知らせ表示
- 銀行からのお知らせを日付付きで一覧表示する

### 4.5 サービス案内
- 主要サービス（住宅ローン、カードローン、口座開設、ネットバンキング）をカード形式で表示

## 5. 非機能要件

- レスポンシブ対応（PC / スマートフォン / タブレット）
- ページ読み込み速度: 3秒以内
- アクセシビリティ: WCAG 2.1 AA準拠
- ブラウザ対応: Chrome、Safari、Edge（最新2バージョン）

## 6. 対象外（今回のスコープ外）

- AIチャット機能
- 音声検索
- 多言語対応
- ユーザー認証・ログイン機能
