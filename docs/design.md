# 技術設計書 — 東京スター銀行 FAQサイト

## 1. 技術スタック

| レイヤー | 技術 | バージョン |
|---------|------|----------|
| フレームワーク | React | 18.x |
| ビルドツール | Vite | 5.x |
| 言語 | TypeScript | 5.x |
| スタイリング | CSS-in-JS（インラインスタイル） | - |
| パッケージ管理 | npm | - |

## 2. ディレクトリ構成

```
src/
├── main.tsx              # エントリーポイント
├── index.css             # グローバルスタイル
└── components/
    └── App.tsx            # メインアプリケーション
```

## 3. コンポーネント設計

### App.tsx（単一コンポーネント）
- ヘッダー（ナビゲーション）
- ヒーローバナー（キャッチコピー + CTAボタン）
- サービスカード（4枚グリッド）
- お知らせ一覧
- FAQセクション（カテゴリ別アコーディオン）
- フッター

## 4. データ管理

- FAQデータ: コンポーネント内にハードコード（将来的にAPI化を検討）
- お知らせデータ: コンポーネント内にハードコード
- 状態管理: React useState（展開中のFAQ管理）

## 5. デプロイ

- ホスティング: Amazon S3 + CloudFront
- ビルド: `npm run build` → `dist/` を S3 にアップロード
- CI/CD: 未構築（手動デプロイ）
